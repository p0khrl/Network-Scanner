# Network Scanner

A simple Python-based network scanner that checks which ports are open on a target system. I built this project while learning about networking and socket programming.

It’s lightweight, runs in the terminal, and lets you scan a custom IP address with a custom port range.

---

## What it does

- Scans a target IP address for open ports  
- Supports custom port ranges (like `1-1000` or `22,80,443`)  
- Shows results directly in the terminal  
- Simple and fast to run  

---

## How to run it

Make sure you have Python installed.

Then run:

```bash
python main.py
